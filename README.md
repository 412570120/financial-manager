## 個人財務管理系統
這是一個專為個人設計的全端收支管理系統，使用者可以透過直觀的介面紀錄每日開銷，並透過動態圖表掌握消費分布。
本專案為課堂實作後自我修改，從前端介面設計到後端 RESTful API 邏輯開發、雲端資料庫串接，皆由個人獨立完成。

--- 
### 專案亮點
1. 全端系統架構：整合 GitHub Pages (前端) 與 Render (後端)，實現跨網域資料互動。
2. 雲端資料管理：串接 Aiven MySQL 雲端資料庫，確保資料持久化與安全性。
3. 動態數據視覺化：整合 Chart.js，將枯燥的數字轉化為直觀的月度消費圓餅圖。

### 相關使用技術
1. 前端:HTML5, CSS3, JavaScript , Chart.js
2. 後端:Node.js, Express.js
3. 資料庫:MySQL (Hosted on Aiven)
4. 部署:GitHub Pages (Frontend), Render (Backend)
5. 版本控制:Git, GitHub

### 核心功能
1. 會員系統：支援使用者註冊、安全登入及登出機制，並使用 localStorage 管理會話狀態。
2. 收支紀錄：支援新增、儲存日期、類型、款項與金額，並即時更新至雲端資料庫。
3. 歷史帳簿：動態讀取資料庫數據，以表格形式呈現歷史收支清單。
4. 月度分析：可切換不同月份，系統自動計算該月收入與支出總計並更新圓餅圖。

### 未來優化

2. 響應式設計：目前針對桌面端優化，未來將進一步適配行動裝置介面。
3. 功能擴充：預計加入「預算超支提醒」與「分類標籤自定義」功能。
4. 介面美化：將介面設計更整潔明瞭，符合使用者觀點。

## 環境變數設定
本專案後端使用 dotenv 管理敏感資訊。
若欲於本地端運行，請於 finance-backend 資料夾下建立 .env 檔案，並填入以下必要資訊：
DB_HOST: 資料庫主機位址
DB_USER: 資料庫使用者名稱
DB_PASSWORD: 資料庫連線密碼
DB_NAME: 資料庫名稱 (預設為 defaultdb)
DB_PORT: 資料庫連線埠 (預設為 24209)
